# Set Environment Variable

* Copy lokasi folder instalasi JDK
* My Computer > Properties > Advanced > Environment Variable
* JAVA_HOME : lokasi folder instalasi JDK
* PATH : tambahkan %JAVA_HOME%\bin
* Klik OK
* Restart Command Prompt
* Test
